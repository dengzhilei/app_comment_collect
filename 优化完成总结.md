## 🎯 核心功能

项目现在专注于两个核心功能：

1. **评论采集**：从 Google Play 采集游戏评论
2. **评论筛选**：智能筛选有价值的评论，输出到文档供AI分析

## 📝 使用方式

### 采集评论
```bash
运行采集.bat "游戏名称" [开始日期] [结束日期]
```

### 筛选评论
```bash
运行筛选.bat "游戏名称"
```

详细说明请查看 `README.md`。
