# 特殊格方案（仅设计，暂不实现）

## 构思
以下为特殊格的设计思路，**不在代码中实现**，待后续需要时再接入。

| 类型     | 名称     | 构思效果 |
|----------|----------|----------|
| start    | 起点     | 每圈经过可给少量奖励或仅作标识 |
| double   | 双倍奖励 | 落格时本次获得数量 ×2 |
| lucky    | 幸运格   | 临时提高 Lv2/Lv3 掉落概率 |
| warp     | 传送格   | 随机传送到另一格（可限制为前/后若干格） |

## 落点建议
- 起点：pathIndex 0
- 双倍：约 1/4 圈处（如 6）
- 幸运：约 1/2 圈处（如 12）
- 传送：约 3/4 圈处（如 18）

实现时再在 config 中增加类似 `SPECIAL_TILES` 的配置，并在落格逻辑中根据格类型分支即可。
