# 🎣 钓鱼棋盘游戏

一个以钓鱼为主题、掷骰子走棋盘格子为主要玩法的休闲游戏。

## 🎮 游戏玩法

### 基本规则

1. **掷骰子移动**
   - 点击"掷骰子"按钮，消耗1个骰子道具
   - 掷出1-6的点数，玩家在棋盘上移动相应格数
   - 初始拥有5个骰子

2. **钓鱼系统**
   - 当玩家移动到标记有🎣的钓鱼点时，可以点击"钓鱼"按钮
   - 钓鱼有成功率，成功后会随机获得不同类型的鱼
   - 鱼的类型包括：小黄鱼、带鱼、鲈鱼、金枪鱼、三文鱼（稀有度递增）

3. **经济系统**
   - 点击"卖鱼"按钮可以卖出所有鱼获获得金币
   - 不同鱼类的价格不同，稀有度越高价格越高
   - 初始拥有100金币

4. **建造系统**
   - 使用金币可以建造各种设施来增强游戏体验：
     - **码头** (200金币)：提高钓鱼成功率
     - **渔具店** (150金币)：降低骰子价格（待实现）
     - **市场** (300金币)：提高鱼获售价50%
     - **加工厂** (500金币)：自动加工鱼获（待实现）

## 🚀 如何运行

1. 直接在浏览器中打开 `index.html` 文件
2. 或者使用本地服务器：
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 使用Node.js
   npx http-server
   ```
3. 在浏览器中访问 `http://localhost:8000`

## 📋 游戏特性

- ✅ 棋盘系统：20格循环棋盘，部分格子为钓鱼点
- ✅ 骰子系统：掷骰子移动，消耗骰子道具
- ✅ 钓鱼系统：在钓鱼点钓鱼，随机获得鱼获
- ✅ 经济系统：卖鱼赚钱，不同鱼类价格不同
- ✅ 建造系统：建造设施增强游戏体验
- ✅ 现代化UI：美观的渐变色彩和流畅的交互
