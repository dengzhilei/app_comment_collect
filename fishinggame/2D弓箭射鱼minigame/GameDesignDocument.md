# 游戏设计文档 (GDD) - 深海猎手

**最后更新**: v2.2 (优化射击手感差异)
**状态**: 创意与玩法规划

---

## 1. 核心概念
**“深海猎手”** 是一款休闲解压的竖版射击游戏。
**核心方针**：**轻策略，重爽感**。

### 1.1 情绪曲线 (Pacing)
1.  **前期 (0-30s)**：捕鱼、积累连击、触发 Fever。
2.  **终局 (30-45s)**：Boss 战，高潮爆发。

## 2. 玩法系统

### 2.1 深度射击机制 (New!)
为了解决“点击与蓄力无区别”的问题，引入**威力 (Power)** 概念。

*   **短按 (Tap) = 速射 (Quick Shot)**
    *   **操作**：按下 < 0.3秒 即松开。
    *   **特点**：射速快，但**威力低**。
    *   **缺点**：射程短，箭矢判定范围小，**无法穿透**任何鱼（只能打最外层）。
    *   **用途**：快速补刀近处的小鱼，维持连击。

*   **长按 (Hold) = 蓄力射击 (Charged Shot)**
    *   **操作**：按住 > 0.3秒，蓄力条上涨。
    *   **特点**：**威力高**。
    *   **满蓄力奖励**：
        1.  **穿透 (Pierce)**：可以穿透 1-2 条小型鱼类。
        2.  **高速**：飞行速度极大提升，几乎瞬间到达（减少预判难度）。
        3.  **击退/打断**：对 Boss 触手造成更高伤害（如 2倍伤害）。
    *   **用途**：打击后排高分鱼（如黄金鲨），或对 Boss 输出。

### 2.2 连击与狂热 (Combo & Fever)
*   **连击**：连续命中 5 次触发 Fever。
*   **狂热模式 (Fever Mode)**：
    *   **效果**：**强制满蓄力**。玩家此时的任何点击都视为“满蓄力穿透箭”，且无视攻速限制。

### 2.3 鱼类生态系统
*   **基础层**：维持 Combo（小丑鱼、金枪鱼）。
*   **奖励层**：时间鱼（+5秒）。
*   **干扰层**：炸弹鱼。

### 2.4 Boss 遭遇战 (The Kraken)
*   **登场时机**：剩余 15秒。
*   **机制**：4条触手。
*   **策略**：必须使用**蓄力射击**才能高效击破触手（速射只有 1点伤害，蓄力有 2-3点）。

## 3. 视觉与反馈美学
*   **蓄力反馈**：蓄力时弓身发光，满蓄力时有“叮”的一声或闪光提示。
*   **箭矢区分**：普通箭矢细小；蓄力箭矢粗大，带拖尾特效。
