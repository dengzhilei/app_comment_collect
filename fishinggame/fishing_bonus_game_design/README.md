# 钓鱼主题 Bonus Game 设计项目

## 项目概述
这是一个游戏设计项目，旨在设计一个类似 Coin Master 或 Monopoly Go 的 bonus game 玩法。

## 核心需求

### 基本结构
- 类似 Coin Master 或 Monopoly Go 的游戏机制
- 踩中特殊格子时触发特殊玩法
- 进行一段演绎后结算奖励
- 类似老虎机的 bonus 玩法或线上 casino 的转盘 bonus game

### 主题要求
- 与**鱼、海洋、钓鱼**其中之一相关联

### 玩法要求
- 需要通过玩法和表现设计营造 **Near Miss** 的感觉
- 即差一点就拿到那个大奖了
- 参考案例：Dice Dream 的玩法（去最终目的地的宝箱，中间有几段路，每次3选1或4选1，顺利就前进一步，失败就结算）
