# 使用说明

## 快速开始

### 1. 安装依赖
双击运行：`安装依赖_智能版.bat`

### 2. 配置游戏ID
编辑 `config.yaml`，填入游戏的App Store和Google Play ID

### 3. 采集数据
双击运行：`运行测试.bat`（测试单个游戏）

或使用配置文件采集：
```bash
python3 -m src.scraper.main_scraper
```

### 4. 筛选评论
双击运行：`运行粗筛.bat`

会生成两个文件：
- **详细版**（Markdown）- 包含完整信息
- **简化版**（TXT）- 可直接复制给AI

### 5. AI分析
将简化版TXT内容复制给AI（ChatGPT、Claude等），让AI进行深度分析

## 工作流程

```
数据采集 → 数据清洗 → 智能筛选 → 输出文档 → AI分析
```

## 核心文件

- `main_simple_filter.py` - 评论筛选主程序
- `src/analyzer/review_filter.py` - 评论筛选器
- `src/scraper/` - 数据采集模块
- `src/processor/` - 数据处理模块

## 可用的批处理文件

| 文件名 | 功能 |
|--------|------|
| `安装依赖_智能版.bat` | 安装Python依赖 |
| `检查依赖.bat` | 检查依赖是否安装 |
| `运行测试.bat` | 测试单个游戏的数据采集 |
| `运行粗筛.bat` | 筛选有意义的评论 |

## 输出文件位置

- **原始数据**: `data/raw/`
- **筛选后的评论**: `output/reports/`
